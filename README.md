# 欢迎使用 xzypy
## 功能特点

- **美观页面**: 简洁优美的页面
- **高亮显示**: 完整的高亮支持
- **密码字典导出**: 支持txt等格式导出
- **响应式设计**: 完美适配各种设备
- **国产中文**: 中文界面

## 快速开始
若要运行此项目请输入:
```cmd
git clone https://github.com/xzy860123/xzypy.git
```
```cmd
cd xzypy
```


```cmd
python xzypy.py
```
